- Node.js
- Express.js
- MongoDB

## 📂 Project Structure
frontend/  → React UI  
backend/   → Node.js API  

## ⚙️ Installation

1. Clone the repository
git clone https://github.com/Sahil-sh378/Sahil-

2. Install dependencies
npm install

3. Start backend
node server.js

4. Start frontend
npm start

## 🎯 Features
- Submit feedback
- View feedback
- Full stack architecture
- MongoDB database storage
